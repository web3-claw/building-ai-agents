## Suggested Learning Path

### Phase 1: Foundations
1. Watch [LLM Introduction](https://www.youtube.com/watch?v=zjkBMFhNj_g) and [LLMs from Scratch](https://www.youtube.com/watch?v=9vM4p9NN0Ts)
2. Read [Understanding Deep Learning](https://udlbook.github.io/udlbook/) (free)
3. Read [Chain-of-Thought Prompting](https://arxiv.org/pdf/2201.11903) paper

### Phase 2: Agent Concepts
4. Watch [Agentic AI Overview (Stanford)](https://www.youtube.com/watch?v=kJLiOGle3Lw)
5. Read [ReAct](https://arxiv.org/abs/2210.03629) and [Reflexion](https://proceedings.neurips.cc/paper_files/paper/2023/file/1b44b878bb782e6954cd888628510e90-Paper-Conference.pdf) papers
6. Read [Building Effective Agents (Anthropic)](https://www.anthropic.com/engineering/building-effective-agents)
7. Read [OpenAI's Practical Guide to Building Agents](https://cdn.openai.com/business-guides-and-resources/a-practical-guide-to-building-agents.pdf)

### Phase 3: Hands-On Building
8. Take [HuggingFace Agents Course](https://huggingface.co/learn/agents-course/en/unit0/introduction)
9. Take [MCP with Anthropic](https://www.deeplearning.ai/short-courses/mcp-build-rich-context-ai-apps-with-anthropic/) course
10. Watch [Building an Agent from Scratch](https://www.youtube.com/watch?v=xzXdLRUyjUg)
11. Explore [GenAI Agents repo](https://github.com/NirDiamant/GenAI_Agents)

### Phase 4: Production & Multi-Agent Systems
12. Take [Multi AI Agent Systems with CrewAI](https://www.deeplearning.ai/short-courses/multi-ai-agent-systems-with-crewai/)
13. Read [AI Engineering](https://www.oreilly.com/library/view/ai-engineering/9781098166298/) by Chip Huyen
14. Take [Evaluating AI Agents](https://www.deeplearning.ai/short-courses/evaluating-ai-agents/) course
